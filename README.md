# Персональный статический веб-сайт

## Описание проекта

Разработан персональный статический одностраничный веб-сайт с использованием языка разметки HTML и каскадных таблиц стилей CSS без применения фреймворков. Веб-сайт реализован в виде одного HTML-файла и содержит структурированную информацию о пользователе. При выполнении работы использованы базовые элементы HTML-разметки и соблюдены правила корректной структуры документа.

## Выполненные требования

В рамках выполнения задания были реализованы следующие требования:

- Создан HTML-документ с корректной базовой структурой (DOCTYPE, html, head, body)
- Указана кодировка документа UTF-8 и заголовок страницы
- На странице размещены:
  - основной заголовок с именем или псевдонимом автора (Мария Кирюшина (DAZZLAXX))
  - текстовое описание (краткая информация о себе)
  - список интересов или навыков
  - изображение (локальное или по URL)
  - гиперссылка на внешний информационный ресурс
- Использованы семантически корректные HTML-теги для оформления содержимого
- Обеспечено корректное отображение страницы в современном веб-браузере

## Достигнутые цели

### 1. Создана корректная структура HTML-документа
- Указан DOCTYPE для определения версии HTML5
- Определена кодировка UTF-8 для корректного отображения кириллицы
- Добавлен заголовок страницы
- Соблюдена иерархия тегов html → head → body

### 2. Реализованы все требуемые элементы контента
- Заголовок с именем автора (Мария Кирюшина (DAZZLAXX))
- Текстовое описание с информацией о себе и направлении обучения
- Структурированный список интересов и навыков
- Изображение (с возможностью загрузки из локальной папки)
- Гиперссылки на внешние информационные ресурсы

### 3. Применены семантические теги
- `<header>` — для шапки сайта
- `<section>` — для логического разделения контента
- `<footer>` — для подвала страницы
- `<ul>`/`<li>` — для представления списка навыков

### 4. Разработан уникальный дизайн в лавандовой цветовой гамме
- Использован градиентный фон (от #f4e9ff до #dcd0ff)
- Подобрана палитра лавандовых оттенков
- Созданы стилизованные кнопки для ссылок
- Добавлены эффекты наведения (hover) для интерактивности
- Применены тени и рамки для визуальной глубины

### 5. Обеспечено корректное отображение в современных браузерах
- Применены относительные единицы измерения (%, max-width)
- Добавлен резервный вариант отображения изображения (fallback)
